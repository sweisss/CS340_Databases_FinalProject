## Quickstart Guide
#### 1) Build DB (Mac/Linux):

```
bash build_db.sh
```

This will delete any existing file named `test.db` and rebuild a new one.

#### 2) Run PHP Server:
```
php -S 127.0.0.1:8000
```

#### 3) Run Web App:
Open a browser and navigate
```
http://127.0.0.1:8000/web_app.php
```
